## Metodoloji (`/nasil-calisiyoruz/` — canlı dil)

1. Metni yapıştırın (≥120 kelime önerilir)  
2. Metin sunucuda işlenir; OZDNA DT motoru çalışır  
3. Tekrar, homojenlik, kelime çeşitliliği, dolgu yoğunluğu ölçülür  
4. Ağırlıklı skor + düşük/orta/yüksek sinyal  
5. İsteğe bağlı: Detaylı Rapor veya Parafraz (PRO+)
