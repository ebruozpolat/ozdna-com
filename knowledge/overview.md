# ozDNA — AI Dedektör & Humanizer API

**ozdna.com**, Findbelow Ventures'ın B2B katmanıdır. OzDNA GPT motoru üzerinden AI metin dedeksiyonu ve humanizasyon REST API olarak sunulur.

## Konumlandırma

- **Hedef:** B2B / Developer / Investor (Phase 1)
- **Segment:** Fintech, hukuk firmaları, üniversite kurumları
- **Durum:** Özel beta — API erişimi davet usulü

## OzDNA GPT

Tüm ürünlerin ortak teknik motoru:

| Ürün | Motor kullanımı |
|------|----------------|
| tezmakale.com | AI dedektörü + humanizer (production) |
| OzDNA Academic | Dedektör + humanizer (Phase 2) |
| OzDNA Comply | Mevzuat analiz motoru (Phase 2–3) |
| ozdna.com | B2B API + dokümantasyon katmanı |

## Dürüstlük notu

- Beta metrikleri (doğruluk, gecikme) Q3 2026'da yayınlanacak
- Production API endpoint'leri backend olgunlaştıkça açılacak
- OpenAPI 3.1 spec: beta erişimi ile birlikte
