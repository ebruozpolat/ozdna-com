# Findbelow Ventures Ecosystem

Multi-layer product stack. All vertical products run on the same ozDNA platform stack; ozdna.com is the external infrastructure (PaaS) layer.

## [tezmakale.com](https://tezmakale.com) — Live · TR

- **Segment:** B2C — Turkey students
- **Role:** Academic panic solver (AI detection, detailed analysis, plagiarism, advising)
- **Proof value:** Live token economics, usage limits, production workload on ozDNA stack
- **URL:** [https://tezmakale.com](https://tezmakale.com) — AI detection and academic workflows (Turkey)

## OzDNA Academic — Phase 2

- **Segment:** B2C/B2B — Global
- **Features:** Detection, citation verification, academic editing
- **Languages:** TR + EN

## ComplyDNA — Phase 2 (Building)

- **Segment:** B2B — TR / EU RegTech
- **Scope:** 5549, BDDK, MASAK, KVKK regulatory monitoring
- **Customers:** Fintech, EMI/PSP, law firms
- **URL:** [https://ozdna.com/products/comply/](https://ozdna.com/products/comply/)
- **Proof value:** Same ozDNA gateway, RAG, and cost engine as [TezMakale](https://tezmakale.com) — vertical depth generic LLM gateways cannot replicate
