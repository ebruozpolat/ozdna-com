# Findbelow Ventures Ecosystem

Multi-layer product stack. All vertical products share OzDNA GPT; ozdna.com is the external infrastructure (PaaS) layer.

> Horizontal LLM gateways optimize calls. ozDNA optimizes vertical AI business economics.

## TezMakale — Live · TR

**Academic AI · Turkey**

AI detection, humanizer, and academic workflows for students under deadline pressure. Live production workload on ozDNA infrastructure — token limits and usage economics in the wild.

| | |
|--|--|
| **Segment** | B2C — Turkey students |
| **Status** | Live |
| **URL** | https://tezmakale.com |
| **Proof value** | Production token economics, per-workflow limits (detect vs humanize), margin discipline before external infra sale |
| **Phase 2 role** | First own-model vertical (**ozDNA Academic 7B**); cost-per-workflow lab; bootstrapped Phase 2a funding gate |

| | |

## Dual proof model (why both matter)

| | TezMakale (live) | OzDNA Comply (Phase 2) |
|--|------------------|------------------------|
| **Proves** | Token economics, routing, margin in production | Vertical depth, RAG governance, enterprise procurement |
| **Funds Phase 2 via** | Cash flow + unit economics | Grants, B2G, bank/fintech pilots |
| **Own-model first ship** | Academic detect / humanize | Regulatory Q&A + citation |
| **Buyer** | Student (B2C) | Compliance officer, legal, risk (B2B) |

TezMakale earns the right to claim infrastructure; Comply earns the right to sell it enterprise.
